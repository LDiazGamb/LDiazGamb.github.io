STRING INDEX

Each character within a string has a location. Each location is represented by a index number, starting from zero (0) for the first location.


FIGURE 1: STRING INDEX EXAMPLE

One way to calculate a particular character index is to substract one (1), for example:

'J'= 1-1 = 0
'A'= 2-1 = 1
'V'= 3-1 = 2
'A'= 4-1 = 3
