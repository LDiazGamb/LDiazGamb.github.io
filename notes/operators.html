OPERATORS

Operators are special symbols or keywords that allow to perform operations with values, variables or expressions. Operators can be of arithmetic, assignment or relational type.

Arithmetic operators perform addition, subtraction, multiplication or division of values or variables.

Operator

Function

+

Addition and string concatenation

-

Subtraction

*

Multiplication

/

Division

TABLE 1: ARITHMETIC OPERATORS

Examples:

int montlySalary;
 
String personName;
 
// annualSalary divided by 12. Returns 20000.
montlySalary = annualSalary / 12;
 
// Returns LEONARDO ALFONZO DIAZ GAMBOA
personName = "LEONARDO ALFONZO" + "DIAZ GAMBOA";               

Assignment operators set the data contained in variables:

annualSalary = 240000;
 
personName = "LEONARDO ALFONZO DIAZ GAMBOA";

Relational operators are used to compare the values or data stored in variables and returns true or false depending of the values and the type of operator used:

Operator

Function

==

Equality

>

Greater than

<

Less than

>=

Greater or equal

<=

Less or equal

TABLE 2: RELATIONAL OPERATORS

Examples:

// Returns true.
240000 / 12 == 20000;
 
// Returns true.
240000 / 12 >= 20000;
 
// Returns false.
240000 / 12 <  20000;
 
// Returns true.
annualSalary > montlySalary;
 
// Returns false.
montlySalary = 0;

Logical operators evaluate conditions and give Boolean results.

Operator

Syntax

Function

AND

&&

Returns true if all of the data in all variables or values are true

OR

||

Returns true if any of the data in all variables or values are true

NOT

!

Returns true if any of the data in all variables or values are false

TABLE 3: LOGICAL OPERATORS

Examples:

int n1 = 5;
 
int n2 = 5;
 
int n3 = 7;
 
int n4 = 3;
 
String n5 = "one";
 
// Returns true.
n1 == 5 && n2 == 5;
 
// Returns true.
n3 > 5 || n4 == 9;
 
// Returns true.
n5 == "one";
