DATA STRUCTURES

Data Structure

Description

Usage

Array

Fixed-size sequential data storage

Used for storing multiple values of the same type

ArrayList

Resizable dynamic array

Preferred when frequent additions/removals are needed

Deque (Double-ended Queue)

Supports insertion/removal at both ends

Useful when both stack and queue properties are needed

HashMap

Key-value pair mapping with fast lookups

Ideal for fast access via unique keys

HashSet

Unordered collection with unique elements

Used for eliminating duplicates in a dataset

LinkedList

Doubly-linked list for efficient insertions and deletions

Best for scenarios requiring frequent modifications

PriorityQueue

Queue with ordered priority

Used in task scheduling and algorithm optimization

Queue

First-In-First-Out (FIFO) structure

Ideal for scheduling tasks and handling requests

Stack

Last-In-First-Out (LIFO) structure

Used in undo operations and function call stacks

TreeSet

Sorted collection with unique elements

Useful when automatic sorting is required
