VARIABLES

Variables are a nominal storage location for specific values or data. To declare a variable, a name and type most be given to it:
 

int annualSalary;
 
String personName;

 
To initialize a variable, a value must be assigned to it:
 

annualSalary = 240000;
 
personName = "LEONARDO ALFONZO DIAZ GAMBOA";

 
To combine the two processes, declaring and initializing a variable, the value can be assigned when the variable is declared:
 

int annualSalary = 240000;
 
String personName = "LEONARDO ALFONZO DIAZ GAMBOA";

 
As a general convention, variables most be given a descriptive name, using camelCase, using letters, numbers and underscores. Variables names cannot start with a number and must not have any special characters.
